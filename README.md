# AllInOne

Monorepo personale pensato per tenere insieme **frontend** e **backend** di una web app di tracciamento attività quotidiane (allenamenti, hobby, alimentazione e altro).

## Struttura

- `frontend/`: interfaccia web (fase iniziale già impostata).
- `backend/`: spazio pronto per API, logica business e persistenza dati (da implementare step-by-step).

## Avvio rapido frontend

Apri `frontend/index.html` direttamente nel browser oppure avvia un server statico locale, ad esempio:

```bash
python3 -m http.server 4173 --directory frontend
```

Poi visita `http://localhost:4173`.
